This code is for the hyperparameter analysis of the proposed RPGL algorithm. 

Use the code HyperParameterRPGL_LP.m and HyperParameterRPGL_LQ.m to generate the results and the results are saved in HyperParameterRPGL_Lp.mat and HyperParameterRPGL_Lq.mat.


Run the code Fig_4_bc.m to get the figure as in paper.
